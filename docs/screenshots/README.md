# Screenshots

Drop screenshots in this folder and they'll show up in the main
[`README.md`](../../README.md). Suggested shots:

| Filename | What to capture |
|---|---|
| `home.png` | The SearchBox home window with the search bar |
| `results.png` | A search with results showing |
| `viewer.png` | A PDF or document open in the built-in viewer |
| `settings.png` | The Settings page |

PNG at roughly 1200px wide is plenty. Once you add one, find the matching
`> 📷 _Screenshot…_` placeholder line in the top-level README and replace it
with, e.g.:

```markdown
![SearchBox home screen](docs/screenshots/home.png)
```
